# Plan 2-1: Repository Manager

## Objective
Create a `RepoManager` class that clones remote GitHub repositories to `~/.sg_agent/workspaces/<repo-name>` using GitPython, and performs `git pull` if the repo already exists locally. This module handles all Git operations with retry logic and user-facing error messages for all failure modes (invalid URL, network errors, permission issues).

## Code Patterns (Model Resilience)
Reference these existing files for code style:
- `agent/config.py` — Match the function structure, docstrings, import style.
- `agent/daemon.py` — Match the retry/backoff pattern from `DaemonWatcher.start()`.

Conventions:
- File naming: `snake_case` (`repo_manager.py`).
- Error handling: Catch `git.exc.GitCommandError`, `git.exc.InvalidGitRepositoryError`, `git.exc.GitCommandNotFound` explicitly. Log + return error info, never crash.
- Logging: Use `logging.getLogger("agent.repo_manager")`.
- Types: Use `pathlib.Path` for all file paths, `typing.Optional`, `typing.Dict`.

<task type="auto">
  <name>Implement RepoManager class</name>
  <files>agent/repo_manager.py</files>
  <action>
    1. Import: `import logging`, `import time`, `from pathlib import Path`, `from typing import Optional, Dict, Any`, `import git` (from `gitpython`).
    2. Set `logger = logging.getLogger("agent.repo_manager")`.
    3. Define constant `WORKSPACES_DIR = Path.home() / ".sg_agent" / "workspaces"`.
    4. Define constant `MAX_CLONE_RETRIES = 3` and `RETRY_DELAY = 5`.
    5. Create class `RepoManager`:
       - `__init__(self, workspaces_dir: Path = WORKSPACES_DIR)`:
         - Store `self.workspaces_dir = workspaces_dir`.
         - Call `self.workspaces_dir.mkdir(parents=True, exist_ok=True)`.
       - `_extract_repo_name(self, url: str) -> str`:
         - Parse the repo name from a GitHub URL.
         - Strip `.git` suffix if present.
         - Use `url.rstrip("/").split("/")[-1].replace(".git", "")`.
         - Return the name string.
       - `clone_or_pull(self, url: str) -> Optional[Path]`:
         - Call `repo_name = self._extract_repo_name(url)`.
         - Set `repo_path = self.workspaces_dir / repo_name`.
         - If `repo_path.exists()` and `(repo_path / ".git").exists()`:
           - Try: `repo = git.Repo(repo_path)`, `repo.remotes.origin.pull()`.
           - Log info: `f"Updated existing repo: {repo_name}"`.
           - Return `repo_path`.
           - Catch `git.exc.GitCommandError as e`: log warning, return `repo_path` anyway (offline mode).
         - Else (clone with retry):
           - Loop `for attempt in range(1, MAX_CLONE_RETRIES + 1)`:
             - Try: `git.Repo.clone_from(url, str(repo_path))`.
             - Log info: `f"Cloned {url} to {repo_path}"`.
             - Return `repo_path`.
             - Catch `git.exc.GitCommandNotFound`: log error `"Git is not installed"`, return `None` (no retry).
             - Catch `git.exc.GitCommandError as e`:
               - If `attempt < MAX_CLONE_RETRIES`: log warning with attempt count, `time.sleep(RETRY_DELAY)`.
               - Else: log error `f"Failed to clone {url} after {MAX_CLONE_RETRIES} attempts: {e}"`, return `None`.
         - Return `None` as fallback.
       - `remove_repo(self, repo_name: str) -> bool`:
         - Set `repo_path = self.workspaces_dir / repo_name`.
         - If not `repo_path.exists()`: log warning, return `False`.
         - Try: `import shutil; shutil.rmtree(repo_path)`.
         - Log info: `f"Removed workspace: {repo_name}"`.
         - Return `True`.
         - Catch `PermissionError as e`: log error, return `False`.
         - Catch `Exception as e`: log error, return `False`.
       - `list_repos(self) -> list[Dict[str, Any]]`:
         - Return a list of dicts for each subdirectory in `self.workspaces_dir` that contains a `.git` folder.
         - Each dict: `{"name": dir.name, "path": str(dir)}`.

    Match style from: `agent/config.py` for function signatures and docstrings.
  </action>
