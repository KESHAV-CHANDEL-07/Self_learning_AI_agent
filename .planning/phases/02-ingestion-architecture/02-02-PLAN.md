# Plan 2-2: Path Filter

## Objective
Create a `PathFilter` class that combines hardcoded default ignore patterns, global config patterns, local config patterns, and a repository's `.gitignore` into a single `pathspec.PathSpec` object. Expose a clean `should_ignore(path: str) -> bool` method for the daemon and other modules to use.

## Code Patterns (Model Resilience)
Reference these existing files for code style:
- `agent/config.py` — Match import style, function signatures, docstrings.
- `agent/daemon.py` line 11 — Currently does a simple `.git` check inline; this module replaces that hardcoded logic.

Conventions:
- File naming: `snake_case` (`path_filter.py`).
- Logging: `logging.getLogger("agent.path_filter")`.
- Types: `List[str]` for pattern lists, `pathlib.Path` for paths.

## Tasks

<task type="auto">
  <name>Implement PathFilter class</name>
  <files>agent/path_filter.py</files>
  <action>
    1. Import: `import logging`, `from pathlib import Path`, `from typing import List, Optional`, `from pathspec import PathSpec`.
    2. Set `logger = logging.getLogger("agent.path_filter")`.
    3. Define constant list `DEFAULT_IGNORE_PATTERNS`:
       ```python
       DEFAULT_IGNORE_PATTERNS = [
           ".git",
           "__pycache__",
           "node_modules",
           ".env",
           "*.pyc",
           ".DS_Store",
           "*.egg-info",
           ".venv",
           "venv",
       ]
       ```
    4. Create class `PathFilter`:
       - `__init__(self, config_patterns: Optional[List[str]] = None, gitignore_path: Optional[Path] = None)`:
         - Initialize `all_patterns = list(DEFAULT_IGNORE_PATTERNS)`.
         - If `config_patterns` is not None: `all_patterns.extend(config_patterns)`.
         - If `gitignore_path` is not None and `gitignore_path.exists()`:
           - Try: open the file, read non-empty/non-comment lines.
           - `with open(gitignore_path, "r", encoding="utf-8") as f:`.
           - `gitignore_lines = [line.strip() for line in f if line.strip() and not line.strip().startswith("#")]`.
           - `all_patterns.extend(gitignore_lines)`.
           - Catch `Exception as e`: `logger.warning(f"Failed to read .gitignore at {gitignore_path}: {e}")`.
         - `self._spec = PathSpec.from_lines("gitwildmatch", all_patterns)`.
         - Log debug: `f"PathFilter initialized with {len(all_patterns)} patterns"`.
       - `should_ignore(self, path: str) -> bool`:
         - Return `self._spec.match_file(path)`.
       - `get_patterns(self) -> List[str]`:
         - Return the combined list of active patterns (for debugging/status output).

    Match style from: `agent/config.py`
  </action>
