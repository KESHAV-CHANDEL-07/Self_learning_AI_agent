# Plan 2-3: Plugin Architecture

## Objective
Create a `PluginRegistry` that auto-discovers plugin files from `~/.sg_agent/plugins/`, validates they expose the required interface (`PLUGIN_NAME`, `PLUGIN_VERSION`, `run(event, context)`), and provides methods to list, load, and execute plugins safely. Built-in plugins (`file_sorter`, `code_cleanup`) will be created as example templates.

## Code Patterns (Model Resilience)
Reference these existing files for code style:
- `agent/config.py` — Match import/function style.
- `tasks/file_sorting.py` — Current task implementation; the plugin version should mirror this logic.
- `tasks/file_cleanup.py` — Current cleanup implementation; will become a plugin example.

Conventions:
- File naming: `snake_case` for module (`plugin_registry.py`), plugin files also `snake_case.py`.
- Logging: `logging.getLogger("agent.plugin_registry")`.
- Error handling: wrap each plugin load in `try/except`, log and skip on failure. Never crash.
- Types: `Dict[str, Any]` for plugin metadata, `Callable` for the run function.

<task type="auto">
  <name>Implement PluginRegistry class</name>
  <files>agent/plugin_registry.py</files>
  <action>
    1. Import: `import logging`, `import importlib.util`, `from pathlib import Path`, `from typing import Dict, Any, Optional, List`.
    2. Set `logger = logging.getLogger("agent.plugin_registry")`.
    3. Define constant `PLUGINS_DIR = Path.home() / ".sg_agent" / "plugins"`.
    4. Create class `PluginRegistry`:
       - `__init__(self, plugins_dir: Path = PLUGINS_DIR)`:
         - Store `self.plugins_dir = plugins_dir`.
         - Create dir: `self.plugins_dir.mkdir(parents=True, exist_ok=True)`.
         - Initialize `self._plugins: Dict[str, Dict[str, Any]] = {}`.
       - `discover(self) -> int`:
         - Clear `self._plugins`.
         - Iterate over `self.plugins_dir.glob("*.py")`:
           - For each `plugin_file`:
             - Call `self._load_plugin(plugin_file)`.
         - Log info: `f"Discovered {len(self._plugins)} plugin(s)"`.
         - Return `len(self._plugins)`.
       - `_load_plugin(self, plugin_path: Path) -> bool`:
         - Try:
           - `module_name = plugin_path.stem`.
           - `spec = importlib.util.spec_from_file_location(module_name, str(plugin_path))`.
           - If `spec is None` or `spec.loader is None`: log warning, return `False`.
           - `module = importlib.util.module_from_spec(spec)`.
           - `spec.loader.exec_module(module)`.
           - Validate: check `hasattr(module, "PLUGIN_NAME")`, `hasattr(module, "PLUGIN_VERSION")`, `hasattr(module, "run")`.
           - If any missing: log warning `f"Plugin {module_name} missing required interface"`, return `False`.
           - Validate: check `callable(module.run)`.
           - If not callable: log warning, return `False`.
           - Store: `self._plugins[module.PLUGIN_NAME] = {"name": module.PLUGIN_NAME, "version": module.PLUGIN_VERSION, "run": module.run, "path": str(plugin_path)}`.
           - Log info: `f"Loaded plugin: {module.PLUGIN_NAME} v{module.PLUGIN_VERSION}"`.
           - Return `True`.
         - Catch `Exception as e`:
           - Log error: `f"Failed to load plugin from {plugin_path}: {e}"`.
           - Return `False`.
       - `run_plugin(self, name: str, event: Dict[str, Any], context: Dict[str, Any]) -> bool`:
         - If `name not in self._plugins`: log warning, return `False`.
         - Try:
           - `self._plugins[name]["run"](event, context)`.
           - Return `True`.
         - Catch `Exception as e`:
           - Log error: `f"Plugin {name} failed: {e}"`.
           - Return `False`.
       - `run_all(self, event: Dict[str, Any], context: Dict[str, Any]) -> Dict[str, bool]`:
         - `results = {}`.
         - For each `name` in `self._plugins`:
           - `results[name] = self.run_plugin(name, event, context)`.
         - Return `results`.
       - `list_plugins(self) -> List[Dict[str, str]]`:
         - Return `[{"name": p["name"], "version": p["version"], "path": p["path"]} for p in self._plugins.values()]`.

    Match style from: `agent/config.py`
  </action>
  <verify>python -c "from agent.plugin_registry import PluginRegistry; pr = PluginRegistry(); pr.discover(); print(pr.list_plugins())"</verify>
  <done>`PluginRegistry` imports cleanly, `discover()` scans the plugins dir without errors, and bad plugin files are logged and skipped.</done>
</task>
